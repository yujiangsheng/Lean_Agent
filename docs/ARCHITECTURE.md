## 核心组件

### 1. ContinuousLearningAgent（持续学习智能体）

**文件**: `src/learning_agent.py`

**职责**:
- 协调知识图谱、经验学习、猜想生成等模块
- 执行学习循环（选择领域 → 推导猜想 → 验证 → 记录）
- 提供统计信息和学习报告

**核心方法**:

```python

```

### 2. UnifiedKnowledgeManager（统一知识库管理器）

**文件**: `src/unified_knowledge.py`

**职责**:
- 管理 11 个数学领域的知识
- 提供知识查询、检索、过滤功能
- 支持跨领域知识关联

**知识层级**:

| 层级 | 枚举值 | 说明 |
|------|--------|------|
| 公理 | `KnowledgeLevel.AXIOM` | 不证自明的基础事实 |
| 定义 | `KnowledgeLevel.DEFINITION` | 数学概念的精确描述 |
| 核心 | `KnowledgeLevel.CORE` | 经典的重要定理 |
| 派生 | `KnowledgeLevel.DERIVED` | 从已有知识推导 |
| 猜想 | `KnowledgeLevel.CONJECTURE` | 待证明的命题 |

### 3. KnowledgeGraph（知识图谱）

**文件**: `src/knowledge_graph.py`

**职责**:
- 管理定理之间的推导关系
- 支持图查询（前驱、后继、路径）
- 持久化存储到 JSON

**数据结构**:

```python
# 节点 (KnowledgeNode)
node = KnowledgeNode(
    id="thm_001",
    statement="∀ a b : ℝ, a + b = b + a",
    statement_cn="加法交换律",
    domain="algebra",
    node_type=NodeType.AXIOM,
    status=NodeStatus.VERIFIED
)

# 边 (推导关系)
# A → B 表示 A 被用于证明 B
graph.add_edge("axiom_add_comm", "thm_sum_symmetric")
```

### 4. ExperienceLearner（经验学习器）

**文件**: `src/experience_learner.py`

**职责**:
- 记录成功的证明经验
- 提取证明模式和特征
- 推荐有效的证明策略

**经验特征**:

```python
features = {
    "has_forall": 1.0,      # 包含全称量词
    "has_exists": 0.0,      # 包含存在量词
    "has_inequality": 1.0,  # 包含不等式
    "statement_length": 45, # 陈述长度
    "num_variables": 3,     # 变量数量
    ...
}
```

### 5. ChainOfThoughtReasoner（多步推理器）

**文件**: `src/learning_agent.py`（内含）

**职责**:
- 执行多步推理（最多 8 步）
- 支持多种推理操作
- 生成推理链并评估质量

**推理操作**:

| 操作 | 说明 | 示例 |
|------|------|------|
| `combine` | 组合两个定理 | A + B → C |
| `specialize` | 特化（代入具体值） | ∀x.P(x) → P(2) |
| `generalize` | 推广 | P(2) → ∀x.P(x) |
| `analogize` | 类比推理 | 平面 → 空间 |
| `contrapose` | 逆否命题 | A→B ⟺ ¬B→¬A |
| `extend` | 扩展推广 | 添加边界情况 |

## 数据流

### 学习循环数据流

```
┌─────────────────────────────────────────────────────────────────┐
│                        学习循环 (Learning Loop)                  │
└─────────────────────────────────────────────────────────────────┘

    ┌──────────────┐
    │  选择领域     │ ←─── 根据经验权重随机选择
    │  (代数/几何/..)│
    └──────────────┘
           │
           ↓
    ┌──────────────┐
    │  获取前置知识  │ ←─── UnifiedKnowledgeManager
    │  (公理+定理)  │
    └──────────────┘
           │
           ↓
    ┌──────────────┐
    │  多步推理     │ ←─── ChainOfThoughtReasoner
    │  生成猜想     │       最多8步推理链
    └──────────────┘
           │
           ↓
    ┌──────────────┐
    │  验证猜想     │ ←─── InductiveVerifier
    │  (数值归纳)   │       大规模采样验证
    └──────────────┘
           │
     ┌─────┴─────┐
     ↓           ↓
┌────────┐  ┌────────┐
│ 成功   │  │ 失败   │
│ 记录   │  │ 丢弃   │
└────────┘  └────────┘
     │
     ↓
┌──────────────┐     ┌──────────────┐
│ KnowledgeGraph│ ←── │ 添加新节点   │
│  知识图谱     │     │ 添加推导边   │
└──────────────┘     └──────────────┘
     │
     ↓
┌──────────────┐     ┌──────────────┐
│ Experience   │ ←── │ 提取特征     │
│ Learner      │     │ 更新权重     │
└──────────────┘     └──────────────┘
```
